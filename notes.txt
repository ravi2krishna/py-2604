    -> Python is Garbage-collected 

        -> Automatically remove "Unused" Data 

            -> Python Automatically cleans up unused data from memory 

                product = "iphone 17"
                price = 120000 
                rating = 4.6

                print(product)

                print(rating)

            -> NOTE: Python cleans up quantity memory, as it's never used in program

    -> Imperative Code With Python 

                def get_user_emails(users):
                        emails = []  # Step-by-step building the result

                        for i in range(len(users)):         # Loop manually
                            user = users[i]                 # Fetch each user

                            if user.get("email") is not None:   # Check condition manually
                                emails.append(user["email"])    # Manually add to list

                        return emails


                    # Sample users list
                    users = [
                        {"name": "Mike", "email": "mike@gmail.com"},
                        {"name": "John", "email": None},
                        {"name": "Sara", "email": "sara@gmail.com"}
                    ]

                    print(get_user_emails(users))


    -> Declarative Code With Python 

            emails = [u["email"] for u in users if u["email"] is not None]
