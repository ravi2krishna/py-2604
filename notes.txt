                def get_user_emails(users):
                        emails = []  # Step-by-step building the result

                        for i in range(len(users)):         # Loop manually
                            user = users[i]                 # Fetch each user

                            if user.get("email") is not None:   # Check condition manually
                                emails.append(user["email"])    # Manually add to list

                        return emails


                    # Sample users list
                    users = [
                        {"name": "Mike", "email": "mike@gmail.com"},
                        {"name": "John", "email": None},
                        {"name": "Sara", "email": "sara@gmail.com"}

    -> Identity Operators

        -> Used To Check if objects are same 

            -> is: Returns True, if objects are same 

            -> is not: Returns True, if objects are not same 

        -> NOTE: 

            ->  (Equality operator ==): Compares the values of two objects. 
                It returns True if their contents are the same, 
                regardless of where they are stored in memory.

            -> is (Identity operator): Compares the identity (memory address) of two objects. 
                It returns True only if both variables refer to the exact same object in memory.    


    -> Bitwise Operator 

        -> Used in Low Level Programming (0101010101)

            -> & - if both the bits are 1, then result is 1 
            -> | - if one of the bits is 1, then reset is 1
