    -> Python is dynamically type-checked and garbage-collected.

        -> Dynamically type-checked

            -> We have a concept called "Data Types" 

                -> Data Types: Categorization Of Data

                    -> text, whole numbers, decimal numbers etc 

            -> NOTE: Dynamically type-checked means, we don't need to explicitly 
            specify Data Types 

                -> product = "iphone 17"
                -> price = 120000 
                -> rating = 4.6
                -> etc 


        -> Static type-checked

             -> NOTE: Static type-checked means, we must explicitly 
            specify Data Types 

                -> String product = "iphone 17";
                -> int price = 120000;
                -> double rating = 4.6;
                -> etc 


    -> Python is Garbage-collected 

        -> Automatically remove "Unused" Data 

            -> Python Automatically cleans up unused data from memory 

                product = "iphone 17"
                price = 120000 
                rating = 4.6
                quantity = 25

                print(product)
                print(price)
                print(rating)

            -> NOTE: Python cleans up quantity memory, as it's never used in program


    -> Python is an "Interpreted" Language 

        -> An Interpreted Language executes line by line 

    -> Java is "Compiled" Language

        -> A Compiled Language translates all the code into machine code and executes 
        all the code at Once 

        -> Compiled Language are FASTER compared to Interpreted Language

    -> NOTE: Performance is impacted Java > Python
